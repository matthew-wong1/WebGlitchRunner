# WebGlitchRunner

A WebGPU program runner (intended to be integrate with WebGlitch)

## Requirements:

Please update the following variables with the relevant paths on your own system:

1. WEBGLITCH_PATH
2. WEBGLITCH_RUNNER_PATH
3. DENO_PATH
4. CHROME_PATH
5. FIREFOX_PATH
6. MACOS_INTERCEPTORS
7. LINUX_INTERCEPTORS
